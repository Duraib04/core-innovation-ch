## Image Specifications

- **Format**: JPG or PNG
- **Dimensions**: Minimum 800x800px (square aspect ratio preferred)
- **File Size**: Under 500KB each (optimized for web)
- **Quality**: High quality, clear product photos

## Tips

1. Use high-quality product photos showing the actual device/interface
2. Include different angles and use cases in the 4 images
3. Ensure good lighting and clear visibility
4. For the chat website, use screenshots of the interface
5. Compress images before uploading to reduce page load time
